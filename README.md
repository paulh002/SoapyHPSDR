# Soapy SDR module for HPSDR and Red Pitaya
This is a experimental HPSDR soapydriver which is still in development
Tested on pavel-demin/red-pitaya-notes with the TRX-DUO

To use this SoapyDriver you have to run the sdr_tranceiver_122_88 on the TRX-DUO
